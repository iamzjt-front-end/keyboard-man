# 第十五天 - vim15

掌握窗口的管理

1.vim窗口管理
  新建窗口  左右: ctrl + w  + v
            上下: ctrl + w + s
  窗口切换  ctrl + hjkl  或者  ctrl + w
  关闭窗口  ctrl + w + c
  只保留当前窗口，关闭其他窗口  ctrl + w + o

2.vscode窗口管理
  新建窗口  左右: ctrl + \
            上下: ctrl + alt + \
  关闭窗口  ctrl + w
            ctrl + k + w
  窗口切换  shift + 方向键(改键实现)
